# Anaya Data Directory and File Naming 

## Data directories'

*clean* indicates naming/spelling matched for dirs and pics


* ancillary: dem, flowline, orthotifs, photos 
* dams_2023: clean 2013-2023 input data, arroyos/pics 
* 2025_survey: clean 2025 input data, arroyos/pics (through October)
* dams_2025_damsep: clean, dam-separated 2025 arroyos/dams/pics, 2025
* dams_all: final output for dams_2025_damsep processed with dams_2013-2023_clean, 
  arroyos/dams/pics
* oldstuff
  * 2025_survey_incomplete: clean 2025 input data, incomplete, only through August
  * analyze: 2022 shapefiles
  * dams_2025_test
  * outdam_2023-05-13
  * outdam_old
* outdam: output logfiles
